# Guia de Instalacion de Power BI Desktop

## Capacitacion ISELIN - Analisis de Datos con Power BI

---

## Opcion 1: Instalacion desde Microsoft Store (Recomendada)

Esta es la forma mas sencilla y garantiza actualizaciones automaticas.

### Pasos

1. Abrir **Microsoft Store** en Windows (buscar "Microsoft Store" en el menu Inicio).
2. En la barra de busqueda, escribir **"Power BI Desktop"**.
3. Seleccionar la aplicacion **Power BI Desktop** (desarrollador: Microsoft Corporation).
4. Hacer clic en **"Obtener"** o **"Instalar"**.
5. Esperar a que se descargue e instale (aproximadamente 400 MB).
6. Una vez instalado, hacer clic en **"Abrir"** o buscarlo en el menu Inicio.

### Primer inicio

1. Al abrir PBI Desktop por primera vez, aparecera una pantalla de bienvenida.
2. Hacer clic en **"Comenzar"** o cerrar la ventana de bienvenida.
3. Iniciar sesion con la **cuenta corporativa de Microsoft 365** (la misma del correo de ISELIN).
   - Esto vincula PBI Desktop con la licencia Pro de la organizacion.
4. Ya esta listo para usar.

---

## Opcion 2: Descarga directa desde el sitio de Microsoft

Si no se tiene acceso a Microsoft Store:

### Pasos

1. Abrir un navegador web.
2. Ir a la pagina oficial: buscar en Google **"Power BI Desktop download"** y seleccionar el resultado de microsoft.com.
3. Hacer clic en **"Descarga gratuita"** o **"Download free"**.
4. Ejecutar el archivo descargado (PBIDesktopSetup_x64.exe).
5. Seguir el asistente de instalacion: Siguiente > Aceptar terminos > Siguiente > Instalar.
6. Esperar a que finalice y hacer clic en **"Finalizar"**.

---

## Verificacion de la instalacion

Para confirmar que todo esta correctamente instalado:

1. Abrir Power BI Desktop.
2. Ir a **Archivo > Opciones y configuracion > Opciones**.
3. Verificar que aparezca la version instalada (la version actual de abril 2026).
4. Ir a **Inicio > Obtener datos > Excel**.
5. Seleccionar cualquier archivo Excel de prueba.
6. Si se abre la ventana de vista previa (Navigator), la instalacion es correcta.
7. Cancelar (no es necesario cargar el archivo ahora).

---

## Verificacion del acceso a Power BI Service

1. Abrir un navegador web.
2. Ir a **app.powerbi.com**.
3. Iniciar sesion con la cuenta corporativa de Microsoft 365.
4. Si se ve el panel de Power BI Service (pagina de inicio con "Home"), el acceso esta confirmado.
5. Si aparece un error de licencia, contactar al area de IT de ISELIN.

## Requisitos del sistema

| Requisito | Minimo |
|---|---|
| Sistema operativo | Windows 10 version 1809+ / Windows 11 / Windows Server 2019+ |
| Procesador | 1 GHz 64-bit (x64) |
| Memoria RAM | 2 GB (recomendado 4 GB o mas) |
| Espacio en disco | 2 GB disponibles |
| Pantalla | 1440 x 900 o superior |
| Conexion a internet | Necesaria para publicar y descargar actualizaciones |
| .NET Framework | 4.7.2 o superior (generalmente ya incluido en Windows 10/11) |

---

## Problemas frecuentes y soluciones

| Problema | Solucion |
|---|---|
| "No puedo instalar desde Microsoft Store" | Usar la Opcion 2 (descarga directa) |
| "Me pide una licencia al abrir" | Iniciar sesion con la cuenta corporativa de ISELIN (no con cuenta personal) |
| "La pantalla se ve muy pequeña" | Ir a Archivo > Opciones > GLOBAL > Seguridad de informe > Ajustar escala |
| "No puedo conectar a un Excel" | Verificar que el archivo no este abierto en Excel al mismo tiempo |
| "Power BI se cierra solo" | Actualizar a la ultima version desde Microsoft Store o descargar nuevamente |
